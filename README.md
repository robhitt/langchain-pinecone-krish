## Methods

- Loading: `PyPDFDirectoryLoader`
- Chunking: `RecursiveCharacterTextSplitter`
- Embedding: `OpenAIEmbeddings`
- Vector DB: `Pinecone`
  - Search Vector DB: `pinecone.similarity_search`
- Model: `OpenAI`
